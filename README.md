# üçä TangerineUI (Formerly web_client)

**The Face of WatermelonD.**

TangerineUI represents the peel/shell of the system. It is designed to be easy to peel (deploy), bright, segmented, and user-friendly.

## Features
- **Reactive Interface**: Visualizes system states (Listening, Thinking, Speaking).
- **Dashboard**: Web controls for system management.
- **Terminal**: Integrated web terminal for direct interaction.

## Installation
This module is automatically configured when installing [WatermelonD](https://github.com/OpenMacedonIA/WatermelonD). To run it independently (Remote Mode), check the main installation guide.
