Install the BerryConnect agent on your satellite devices to get started

Setup Instructions

    1. Click "Generate Installer Script" above to download a pre-configured installer
    2. Copy the entire BerryConnect/PiZero folder to your satellite Raspberry Pi
    3. Run chmod +x install.sh && ./install.sh on the satellite device
    4. The agent will auto-discover the broker and start sending telemetry
    5. Agents will appear here automatically once connected

    Manual Configuration

    If auto-discovery fails, create config.json manually:

    {
  "broker_address": "your.broker.ip",
  "broker_port": 1883,
  "agent_id": "AUTO",
  "telemetry_interval": 10,
  "log_level": "INFO"
}
